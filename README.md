
# Planner Turni Apple Style

## Come avviare

1. Apri la cartella in VS Code
2. Installa estensione Live Server
3. Avvia index.html

## Pubblicazione GitHub Pages

1. Crea repository GitHub
2. Carica tutti i file
3. Settings → Pages
4. Branch: main
5. Save

Dopo pochi secondi avrai il link della tua web app.

## Funzioni previste

- Vista Apple Calendar
- Vista annuale/mensile/settimanale
- Regole Rep/Frep
- Conteggi automatici
- Inserimento rapido periodi
- Installabile su iPhone
